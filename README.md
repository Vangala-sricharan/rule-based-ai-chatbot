# Rule Based AI Chatbot (Python)

A simple AI Chatbot developed using Python and Dictionary-Based Responses.

## Features

- Greetings Support
- AI Related Responses
- Python and C++ Queries
- Continuous Chat Loop
- Exit Command
- Fallback Responses

## Technologies Used

- Python
- Dictionary Data Structure
- Loops
- Conditional Statements

## Sample Commands

- hello
- hi
- ai
- python
- c++
- help
- exit
